## Description

#### Backend for parcel out chatbot 💬

<br>

## Installation

```bash
$ npm install
```

<br>

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

<br>

## DataModel
#### 🏗 User (=Builder)
  - id
  - createdAt
  - updatedAt
  - name
  - phoneNumber
  - password
  - role
  - customers
  - customerId

#### 👤 Customer
  - id
  - createdAt
  - updatedAt
  - name
  - phoneNumber
  - users
  - userId
